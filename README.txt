This tool can be used to import BDOT10k shapefiles or GML files and adds them in the QGIS canvas. Also, layes can be symbolized to be similar to the topographic map in 1:10 000 scale.

To import BDOT10k GML or SHP files you have to select folder with BDOT10k files in GML or Esri Shape Files format.
Example of BDOT10k GML or Esri Shape Files names: PL.PZGiK.238_2476_OT_SKRW_P.shp
